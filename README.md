# Credit-Risk-Analysis

This project is an end-to-end Machine Learning solution designed to predict the probability of loan default. It features a predictive backend trained on real-world lending data and an interactive Streamlit dashboard for real-time risk assessment.

Business Problem
Banks and financial institutions face significant losses due to non-performing assets (NPAs). 

This project aims to:

* Identify high-risk borrowers before loan approval.

* Provide a transparent "Risk Score" to assist credit officers.

* Understand the key drivers (features) that lead to loan defaults.

Technical Stack :

* Language: Python 3.x

* Libraries: Pandas, Scikit-Learn, XGBoost, Joblib

* Visualization: Plotly, Seaborn

* Deployment: Streamlit
